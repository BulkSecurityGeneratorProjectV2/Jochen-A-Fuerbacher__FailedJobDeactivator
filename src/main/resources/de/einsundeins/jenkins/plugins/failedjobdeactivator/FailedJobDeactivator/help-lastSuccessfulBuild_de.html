Die Zeit in Tagen, die hier eingestellt wird, ist die Zeitdifferenz von heute aus, nach denen die Detektion aktiv werden soll. Sie bezieht sich auf den Zeitpunkt, an dem der Job das letzte Mal erfolgreich gebaut wurde.
Bei Eingabe von ungültigen Werten, wird der Standardwert übernommen.

Ungültige Werte sind:
- 0
- negative Werte
- keine Zahlen
